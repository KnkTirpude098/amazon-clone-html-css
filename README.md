# Amazon Clone (HTML & CSS)

A fully responsive Amazon homepage clone built using pure HTML5 and CSS3.  
This project recreates the look and feel of Amazon’s landing page with a modern UI layout, responsive sections, navigation bar, product cards, and footer.

---

## 🚀 Live Demo

https://knktirpude098.github.io/amazon-clone-html-css/

---

## 📌 Features

- Responsive Amazon-inspired layout
- Navigation bar with search section
- Hero banner section
- Product cards and shopping sections
- Footer with multiple panels
- Clean UI using Flexbox and CSS styling
- Mobile responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Design
